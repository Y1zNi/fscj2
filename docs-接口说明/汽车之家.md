# 汽车之家（接口字段说明）

---

## 1. 接口一：作者主帖列表 JSON

**`POST https://i.m.autohome.com.cn/topic/AjaxTopic`**

**Query**：`pageIndex`、`pageSize`、`uid`。**Body**：`application/x-www-form-urlencoded`，常见 `v=<毫秒时间戳>`。

**结果**：`returncode === 0`；`result` 有 `pageindex`、`pagecount`、`rowcount`、`list`。

**`result.list[]`**

| 字段 | 说明 |
|------|------|
| `topicid` | 帖子 id |
| `title` | 标题 |
| `topictype` | 列表上类型文案，如图 |
| `postdate` | 发帖时间 |
| `lastreplydate` | 最后回复时间 |
| `replycount` | 回复数 |
| `bbsid` | 论坛数字 id |
| `bbs` | 论坛字母，拼进帖链 `thread-{bbs}-{bbsid}-…` |
| `bbsname` | 论坛名称，如湖南 |
| `ispic` | 是否带图类标记 |
| `ispoll` | 是否投票帖类标记 |
| `ishtml` | 是否 html 帖类标记 |
| `isrecommend` | 是否编辑推荐类（以返回值为准） |
| `isrefine` | 是否精华类（以返回值为准） |
| `isdelete` | 是否删除 |
| `isclose` | 是否关闭 |

## 2. 接口二：M 站帖子详情 HTML（旧版）

**`GET https://club.m.autohome.com.cn/bbs/thread-{bbs}-{bbsid}-{topicid}-1.html`**  
例：`/bbs/thread-a-100014-113412569-1.html`。`Host: club.m.autohome.com.cn`。

返回 **HTML**，无 `__NEXT_DATA__`。主从内联 **`var tz`**、**`Config.share`**、DOM。

### 2.1 `tz.bbs`

| 字段 | 说明 |
|------|------|
| `Bbs` | 论坛字母 |
| `BbsId` | 论坛数字 id |
| `BbsName` | 论坛名称 |
| `Moderators` | 版主 uid 数组 |

### 2.2 `tz.topic`

| 字段 | 说明 |
|------|------|
| `TopicId` | 帖子 id |
| `Title` | 标题 |
| `MemberId` | 作者 uid |
| `tDate` | 发帖时间 |
| `TopicType` | 如图 |
| `tsource` | 来源标识 |
| `IsPoll` | 投票相关 |
| `Delete` | 删除标记 |
| `IsCloseReply` | 是否关回复 |
| `club_topiccards_url_list` | 卡片 / 封面图 URL 列表 |

### 2.3 `Config` 与其它脚本

| 名 | 说明 |
|----|------|
| `Config.share.id` | 帖子 id |
| `Config.share.title` | 标题 |
| `Config.share.url` | PC 帖链（常含 hex） |
| `Config.share.description` | 短摘要 |
| `Config.share.img` | 分享图 |
| `Config.preload_img` | 预加载图，多为首图 |
| `url_302` | 相对路径，可从中解析 hex |
| `tz.model` | `topicid`、`ispoll`、`videoId`、`isask`、`PageIndex`、`RowCount` 等 |

### 2.4 DOM

| 要取的 | 位置 |
|--------|------|
| 标题 | `.bbs-post-title h1` |
| 正文容器 | `#content.conttxt` |
| 正文段落 | `.tz-paragraph` |
| 图 | `.tz-picture img`，优先 `data-src-webp` |
| 帖子 id | `section.bbs-post-main` 的 `data-id` |
| 作者 uid | 同上 `data-memberid` |
| 时间串 | 同上 `data-time` |
| 昵称 | `a.user-name` 文本 |
| 作者 uid（链） | `a.user-name` 的 `href` 中数字 |
| 头像 | `.user-face img[src]` |
| 浏览 | `#topicnumdata`（首包常占位） |
| 兜底 | `a.bbs-report` 的 `href` query：`objectid`、`authorid`、`url`、`releasetime` 等 |

### 2.5 本页常见二次请求（不在首包 HTML 体里）

| 接口 | 用途 |
|------|------|
| `https://maindatahot.autohome.com.cn/hotdata/get_hot_data?...` | pv、回复、vv、点赞等 |
| `https://club-open-api.autohome.com.cn/topic/forward?topicId=...&_appid=club.m` | 原文卡片 |
| `https://la.autohome.com.cn/club/club_get_vote_info_by_topic_id?...` | 投票 |
| `https://la.autohome.com.cn/m/proxy_club_get_hot_score_by_topicids?...` | 热门标 |
| `/adaptive/forum/GetTopicListPartial?...` | 相关推荐 |
| `/common/api/club/GetOrderedCategories?...` | 分类分页 |

## 3. 接口三：M 站帖子详情 HTML（Next.js）

**`GET https://club.m.autohome.com.cn/bbs/thread/{hex}/{topicid}-1.html`**  
例：`/bbs/thread/9494b1f812652465/113412569-1.html?pvareaid=101968`。`Host: club.m.autohome.com.cn`；`Referer` 可能为旧式 `thread-a-…` 链。

返回 **HTML**，首包含 **`<script id="__NEXT_DATA__" type="application/json">`**。解析 JSON 取 **`props.pageProps`**。

### 3.1 `mainTopic`

| 字段 | 说明 |
|------|------|
| `title` | 标题 |
| `subtitle` | 副标题 |
| `t_content` | 正文整段 HTML（含 `tz-paragraph`、图等） |
| `topiccards` | 卡片数组：`topicid`、`url`、`ctype`、`des`、`otherattributes`（宽高、`linkurl` 等） |
| `tdate` | 发帖时间 |
| `views` | 浏览数 |
| `topictype` | 如图 |
| `bbs` | 论坛字母（与 URL 中 hex 并存） |
| `bbsid` | 论坛数字 id |
| `bbsname` | 论坛名称 |
| `tclose` | 是否关帖 |
| `tdelete` | 是否删除 |
| `tispoll` | 投票相关 |
| `ispic` | 是否带图 |
| `isrefine` | 是否精华 |
| `t_lasteditdate` | 最后编辑时间 |
| `tsource` | 来源标识 |
| `issolve` | 问答相关 |
| `solveid` | 问答 id |
| `iscard` | 是否卡片帖等 |
| `tag` | `{ tagId, tagName }[]` |
| `isjingxuan` | 是否精选 |
| `iseditorpushjingxuan` | 编辑推送精选 |
| `jingxuanname` | 精选名称 |
| `iseditorpushindex` | 编辑推送首页等 |
| `pubprovincename` | 发布省 |
| `pubcityname` | 发布市 |
| `category` | 分类（可为 null） |
| `catalog` | 目录（可为 null） |
| `piccount` | 图数量（可为 null） |
| `videoinfo` | 视频信息（可为 null） |
| `firstPic` | 首图（可为 null） |

### 3.2 `pageProps` 其它

| 路径 | 说明 |
|------|------|
| `query` | `topicId`、`pageIndex`、`pvareaid` |
| `pvTrack` | `site`、`category`、`subcategory`、`object` |
| `isNotData` | 是否无数据 |

### 3.3 DOM

| 要取的 | 位置 |
|--------|------|
| 论坛名（导航） | `nav` 内指向 `forum-a-{bbsid}-1.html` 的 `h1` |
| 标题 | `.topic-title h1` |
| 正文 | `.topic-content .tz-paragraph` |
| 图 | `.topic-content .tz-picture img`：懒加载看 **`data-src`**；完整以 `t_content` / `topiccards` 为准 |

### 3.4 页内二次请求

与 **§2.5** 同类接口（pv、原文卡片、投票、热门标、相关推荐、分类等）；Next 是否发起、URL 是否完全一致以实际 **Network** 为准。
