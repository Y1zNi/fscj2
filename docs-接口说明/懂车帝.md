# 懂车帝（接口字段说明）

---

## 1. 接口一：用户主页 HTML

**`GET https://m.dongchedi.com/user/{userId}`**  
也可由 `https://www.dongchedi.com/user/all/{userId}` 转成 H5 后请求。

返回 **HTML**，首包含 **`<script id="__NEXT_DATA__" type="application/json">`**。  
主取路径：`__NEXT_DATA__.props.pageProps`。

### 1.1 用户信息（`pageProps.userInfo.info`）

| 字段 | 说明 |
|------|------|
| `user_id` / `user_id_str` | 用户 id |
| `name` / `screen_name` / `nick_name` | 昵称（多字段兜底） |
| `avatar_url` | 头像 |
| `background_img_url` | 背景图 |
| `fans_num` | 粉丝数 |
| `counters[]` | 统计项（粉丝/关注/获赞等，含 `title`、`count`） |
| `medal_list[]` | 勋章（名称/图标/描述） |
| `join_info` | 例如驾龄天数等 |

### 1.2 内容列表（`pageProps.defalutTabData.data[]`）

> 说明：字段会随内容类型变化（文章/视频/口碑等），以下是常见可用字段。

| 字段 | 说明 |
|------|------|
| `gid` / `gid_str` | 内容 id（常用） |
| `group_id` / `group_id_str` | 内容 id 另一套命名 |
| `title` | 标题 |
| `content` / `abstract` | 摘要 |
| `display_time` | 发布时间（时间戳） |
| `read_count` | 阅读/播放 |
| `comment_count` | 评论数 |
| `digg_count` | 点赞数 |
| `article_type` / `content_type` | 内容类型 |
| `share_info.share_url` | 分享链接（可反解 id） |
| `image_list[]` / `detail_image_list[]` | 图片列表 |
| `video_info` | 视频结构（视频类） |
| `user_info` | 作者信息（卡片里会带） |

### 1.3 列表字段补充说明

| 项 | 规则 |
|----|------|
| 今日判断 | `display_time` 转本地日期，与当天比较 |
| 内容 id | 优先 `gid_str/gid`，再 `group_id_str/group_id`，再从 `share_url` 提取 |
| 去重键 | `article_id`（或统一后的内容 id） |

## 2. 接口二：文章详情 HTML

**`GET https://m.dongchedi.com/article/{articleId}`**

返回 **HTML**，首包含：
- `__NEXT_DATA__`
- `application/ld+json`（通常 `@type = NewsArticle`）

### 2.1 主结构（`pageProps.articleDetail`）

| 字段 | 说明 |
|------|------|
| `title` | 标题（主来源） |
| `content` | 正文 HTML（主来源） |
| `abstract` / `summary` | 摘要 |
| `publish_time` | 发布时间（时间戳） |
| `content_publish_time` | 发布时间兜底（时间戳） |
| `watch_count` / `read_count` | 阅读数 |
| `comment_count` | 评论数 |
| `digg_count` | 点赞数 |
| `article_type` / `sub_article_type` | 内容类型 |
| `url` | 文章原始 URL（如 toutiao group 链） |
| `media_user` | 作者信息（id、昵称、粉丝、关注等） |
| `cover_image_info.image_list[]` | 封面图 |
| `motor_material_car_info` | 关联车系/价格/评分等（汽车垂类） |
| `relation_series_map` | 关联车系 map |
| `topic_tag_list[]` | 标签 |
| `material_entrance_list[]` | 素材入口（如用车口碑） |
| `have_essence` | 是否精选（常见 0/1 或布尔） |

### 2.2 `ld+json`（`NewsArticle`）可提取

| 字段 | 说明 |
|------|------|
| `headline` | 标题兜底 |
| `datePublished` | 发布时间兜底（ISO） |
| `author[].name` | 作者名 |
| `author[].url` | 作者主页 |
| `image[]` | 封面图 |

## 3. 抓取注意事项（通用）

| 项 | 建议 |
|----|------|
| Host | 移动端优先 `m.dongchedi.com` |
| Header | `User-Agent`、`Referer` 建议带上 |
| Cookie | 常规可不带；遇风控再加可选 cookie 透传 |
| 兼容 | `defalutTabData` 与 `defaultTabData` 都建议兼容 |
| 文案 | `content` 为 HTML，建议清洗后再落盘 |
